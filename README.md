# thumblify-MERN-backend

A scalable backend API powering the AI Image Generator application, this service handles image generation requests, user sessions, prompt processing, and data persistence for the frontend application.

## Features

- AI image generation using Gemini
- Secure session-based authentication
- Prompt validation and processing
- Centralized error handling

## Tech Stack

- Node.js/Express.js
- TypeScript
- Express Session
- Gemini SDK
- MongoDB

## API Responsibilities

- Receive image generation requests from the frontend
- Communicate with the Gemini API
- Manage user sessions and authentication
- Store prompts and generated image metadata
- Return generation results to the client

## Frontend Repository

This API is designed to work with the React frontend application available in the `thumblify-MERN-frontend` repository.
